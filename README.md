# Burger Project

A simple web project built for learning **HTML, CSS, and JavaScript**. This project is part of a study session to practice building interactive web pages.

## Features

- Responsive layout using HTML and CSS
- Interactive elements using JavaScript
- Simple navigation (Home, Menu, Cart, Login)
- Add-to-cart functionality (basic implementation)
- Study-focused structure for learning purposes

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
